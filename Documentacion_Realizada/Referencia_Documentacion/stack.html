Stack Tecnológico v2.0

Selección de herramientas para una arquitectura escalable, segura y offline-first.

Justificación de la Arquitectura PWA

Se ha seleccionado el modelo PWA por su despliegue instantáneo, bajo costo de mantenimiento y excelentes capacidades offline mediante Service Workers.

Comparativa Tecnológica

Criterio PWA (Seleccionado) Nativo
Tiempo Desarrollo Bajo (3 meses) Alto (6+ meses)
Costo Licencias $0 $198/año
Facilidad Uso Muy Alta (URL) Media (Tiendas)

Frontend (Captura en Planta)

  • Framework: React 18 + TypeScript (Vite).
  • Persistencia: Dexie.js (IndexedDB).
  • Hardware: html5-qrcode para escaneo.

Backend (API REST)

  • Runtime: Node.js (LTS).
  • Framework: Fastify (Alto rendimiento).
  • ORM: Prisma.

Base de Datos

PostgreSQL 15+: Uso de JSONB para auditoría ALCOA+ e índices B-Tree para optimización de búsquedas.

Estrategia de Despliegue

Orquestación mediante Docker Compose para asegurar consistencia entre entornos de desarrollo, staging y producción.
